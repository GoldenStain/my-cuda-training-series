## 小笔记

### lecture 3

sp unit(cores)计算单精度，ld unit计算双精度。

CPU的概念映射到GPU中，并不是cores，GPU的core更类似于cpu当中的ALU，功能甚至比alu更少

gpu中和CPU的core相对应的概念是SM(streaming multiprocessor).
